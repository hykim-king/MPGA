#### MakeProgramGreatAgain(MPGA)조

### 00. 프로젝트 목표  
+ OracleDataBase를 활용하여 DB를 개발한다.  
+ Spring MVC패턴으로 Back-End/jsp, Bootstrap로 Front-End를 개발하여 Full-stack 개발 기술을 습득한다.  
+ Github로 형상관리를 경험해보고 직무수행에 필요한 능력을 기른다.
